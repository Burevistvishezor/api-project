# 🔐 REST API with JWT Authentication

## 📌 Description
This is a simple REST API built with Flask.  
It supports user authentication using JWT tokens and task management.

## ⚙️ Technologies
- Python
- Flask
- Flask-SQLAlchemy
- Flask-JWT-Extended
- SQLite

## 🚀 Features
- User registration
- User login
- JWT authentication
- Protected routes
- Create tasks
- Get user tasks

## 🔑 Authentication
After login, you receive a JWT token.

Use it in headers:
Authorization: Bearer <your_token>

## 📡 API Endpoints
